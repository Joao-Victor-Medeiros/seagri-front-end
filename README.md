# mobile-map-form-flow

Aplicação mobile para formulário com mapa, construída com Vite, React, TypeScript e Tailwind CSS.

## Como executar

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev`: inicia o ambiente de desenvolvimento
- `npm run build`: gera a versão de produção
- `npm run test`: executa a suíte de testes
